# PROMPT สำหรับสร้างรูป Rich Menu Line OA

## 📝 Prompt ภาษาไทย (สำหรับ Nana Banana / AI Image Generator)

```
สร้างรูปภาพ Rich Menu สำหรับ Line Official Account ธุรกิจประกันรถยนต์

**ขนาด:** 1200 x 810 pixels (แนวนอน)
**สไตล์:** ทันสมัย มืออาชีพ น่าเชื่อถือ สีสันสดใสแต่ไม่เวอร์

**โครงสร้าง (3 ปุ่ม):**

**ปุ่มที่ 1 - บนสุด (ใหญ่):**
- ขนาด: เต็มความกว้าง ความสูงประมาณ 60% ของรูป
- พื้นหลัง: ไล่สีจากสีส้ม-แดง (Orange-Red Gradient) #e85a3d ไป #d54a2e
- ไอคอน: รูปรถยนต์ 🚘 (สีขาว ขนาดใหญ่)
- ข้อความหลัก: "เช็คเบี้ย / ซื้อประกัน" (สีขาว ตัวหนา ขนาดใหญ่)
- ข้อความรอง: "ไปที่ 724 ทันที →" (สีขาวอ่อน ตัวปกติ)
- Effect: มีแสงเงาเล็กน้อย ดูมีมิติ

**ปุ่มที่ 2 - ล่างซ้าย (เล็ก):**
- ขนาด: ครึ่งซ้ายของแถวล่าง
- พื้นหลัง: ไล่สีฟ้า-น้ำเงิน (Cyan-Blue Gradient) #0891b2 ไป #0e7490
- ไอคอน: กระเป๋าหรือเอกสาร 💼 (สีขาว)
- ข้อความหลัก: "สมัครสมาชิก" (สีขาว ตัวหนา)
- ข้อความรอง: "ร่วมธุรกิจกับเรา" (สีขาวอ่อน)

**ปุ่มที่ 3 - ล่างขวา (เล็ก):**
- ขนาด: ครึ่งขวาของแถวล่าง
- พื้นหลัง: ไล่สีม่วง-น้ำเงิน (Purple Gradient) #7c3aed ไป #6d28d9
- ไอคอน: คนยกมือ 🙋‍♂️ หรือหูฟัง (สีขาว)
- ข้อความหลัก: "ติดต่อเจ้าหน้าที่" (สีขาว ตัวหนา)
- ข้อความรอง: "พร้อมให้คำปรึกษา" (สีขาวอ่อน)

**รายละเอียดเพิ่มเติม:**
- แต่ละปุ่มมีขอบมนเล็กน้อย (rounded corners)
- มีเส้นแบ่งสีขาวบางๆ ระหว่างปุ่ม
- ตัวอักษรใช้ฟอนต์ที่อ่านง่าย ไทย-อังกฤษ (เช่น Kanit, Sukhumvit หรือ Prompt)
- ภาพรวมดูเป็นระเบียบ สมดุล
- ไม่มีพื้นหลังโปร่งใส (solid background)

**โทนสีหลัก:**
- น้ำเงินเข้ม (Navy Blue): #1e4a8c
- ส้ม (Orange): #e85a3d
- ฟ้า (Cyan): #0891b2
- ม่วง (Purple): #7c3aed

**Reference:** ดูน่าเชื่อถือเหมือนแอพธนาคาร แต่สีสันสดใสกว่าเล็กน้อย
```

## 📝 Prompt ภาษาอังกฤษ (ถ้าใช้ Midjourney/DALL-E)

```
Create a Line Rich Menu image for car insurance business, 1200x810 pixels horizontal layout.

**Layout:**
1. TOP BUTTON (60% height): Orange-red gradient (#e85a3d to #d54a2e) with white car icon 🚘, text "เช็คเบี้ย / ซื้อประกัน" in bold white Thai font, subtext "ไปที่ 724 ทันที →"

2. BOTTOM LEFT (20% height): Cyan-blue gradient (#0891b2 to #0e7490) with briefcase icon 💼, text "สมัครสมาชิก" in white

3. BOTTOM RIGHT (20% height): Purple gradient (#7c3aed to #6d28d9) with person raising hand icon 🙋‍♂️, text "ติดต่อเจ้าหน้าที่" in white

**Style:** Modern, professional, trustworthy, banking app aesthetic but more colorful, rounded corners on buttons, thin white borders between sections, gradient backgrounds, clean typography.

**Colors:** Navy blue, orange, cyan, purple - vibrant but professional.
```
