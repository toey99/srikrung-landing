📷 แปลด้วยการถ่ายรูป (45 ภาษา)
  1. เลือก "การแปลด้วยถ่ายรูป"
  2. เลือกภาษาต้นทาง → ปลายทาง
  3. กดปุ่มกล้อง ถ่ายข้อความ
  4. เครื่องแปลให้ทันที
เหมาะสำหรับ: เมนูอาหาร, ป้ายบอกทาง, เอกสาร
🎙️ บันทึกเสียงพร้อมแปล
  1. เลือก "บันทึกเสียง"
  2. เลือกภาษาจีนหรืออังกฤษเป็นต้นทาง
  3. กดไมค์ พูด → กดหยุด → บันทึก
  4. ฟังย้อนหลังที่รายการบันทึก
⚙️ การตั้งค่า (การตั้งค่า → ...)
WLANเชื่อมต่อ WiFi / Hotspot Bluetoothเชื่อมหูฟัง (ฟังย้อนหลัง)
นอนหลับตั้งเวลาดับจอ (30วิ/1นาที/3นาที/ไม่ดับ) ภาษาระบบเลือกภาษาเมนู (12 ภาษา)
ล้างข้อมูลลบประวัติการแปล อัพเดทตรวจสอบเวอร์ชั่นใหม่
